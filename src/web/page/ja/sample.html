サンプルアプリ

Hello World アプリ

PhoneGap を使用したデモアプリです。デバイス側の複数の機能 ( カメラ、連絡帳など ) にアクセスしています。

RSS リーダーのサンプル

jQuery Mobile を使用した、RSS リーダーのサンプルアプリです。



メモ帳アプリ

jQuery Mobile と localStorage を使用した、簡単なメモ帳アプリです。


ブロック崩しゲーム

パドルを操作して、ボールを落とさないように打ち返して、ブロックを崩していくゲームです。


Twitter クライアント

Twitter のタイムラインを表示するサンプルアプリです。



Facebook クライアント

Monaca から　Facebook に接続を行い、友達一覧を表示するサンプルアプリです。Onsen UI と InAppBrowser を使用します。

バックエンド メモ帳アプリ

Monaca Backend API を使用したメモ帳アプリを作成できる、サンプルアプリです。



Inmobi クライアント

モバイルアド ( 携帯端末への広告配信ネットワーク ) を Monaca に組み込んだアプリです。ここでは、モバイルアドとして、Inmobi を使用します。

Flickr サンプル

Flickr を使用したサンプルアプリです。Flickr から写真を取得して、スライドショーで表示します。



TODO 管理アプリ

TODO 管理のサンプルアプリです。TODO の一覧表示と TODO の登録をすることができます。

電車図鑑アプリ

電車図鑑のサンプルアプリです。東海道から東北区間で運行されている車両を表示します。

時計アプリ

簡単なアニメーションを使用して、現在の日付と時間を表示する時計アプリです。

誕生年の計算アプリ

誕生年を計算するアプリです。


おみくじ占いアプリ

おみくじ占いができるアプリです。

iBeacon

iBeaconデバイスの基本的な利用方法を扱うサンプルアプリ。

テンプレート

最小限のテンプレート

Monaca アプリ作成時に使用する、最小限のテンプレートです。



Onsen UI 最小限のテンプレート

Onsen UI アプリ用の最小限のテンプレートです。Onsen UI が持つ機能をいくつか使用しています。


Onsen UI スライディングメニュー

Onsen UI のスライディングメニューとナビゲーションを使用しています。

Onsen UI ナビ付きタブバー

Onsen UI を使用したアプリに、タブバーを組み込んでいます。
