Native Components
========================

Monaca has Native Functions that provide Native UI and behaviors. With Native Functions, the Monaca app is able to have Native UI and Menu components.


You can create cross-platform applications with each OS's own screen parts. You can create an application with smoother textures while maintaining the cross-platform feature.
